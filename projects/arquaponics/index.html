Arquaponics is a project that I did with my fellow students in the minor Web Development (HvA). This project is build in VueJS and used AR functionality. What we created was a datavisualisation with D3JS in AR. We did this project for De Ceuvel They have a sustainable planned workplace for creative and social enterprises on a former shipyard on the Johan van Hasselt kanaal off the river IJ in Amsterdam North. So we wanted to show the data as interactivly as we can.

Within this project I worked on D3.js and AR. And worked in VueJS for the first time. The project was a success and there were great things started from this project. Starting a collaboration group called Bowtyi.
